# SRINIVASA COOKING AND CATERING - Premium Website

This is a premium, responsive static website for **Srinivasa Cooking and Catering**.

## Setup & Running
1. Open `index.html` in any modern web browser.
2. Ensure you have an internet connection for Tailwind CSS, AOS, and Font Awesome CDNs to load.
3. The animations and images will load automatically.

## Project Structure
- `index.html`: The Home page with hero and services.
- `menu.html`: Separate interactive menu page.
- `plans.html`: Detailed catering plans page.
- `about.html`: Company values and founder information.
- `contact.html`: Booking form and contact details.
- `index.css`: Shared premium styling for all pages.
- `assets/`: High-quality media assets.

## Features
- **Black and Gold Theme**: A luxurious color palette for a premium brand feel.
- **AOS Animations**: Modern scroll animations on every section.
- **Dynamic Menu**: JavaScript-powered tab switching for different cuisine categories.
- **Glassmorphism Nav**: A modern, translucent navigation bar.
- **Responsive Design**: Optimized for Desktop, Tablet, and Mobile.
